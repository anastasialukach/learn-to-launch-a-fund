## Glossary

- **LP (Limited Partner)** - Passive investor in the fund
- **GP (General Partner)** - Active fund manager
- **Carried Interest (Carry)** - GP's share of fund profits (typically 20%)
- **Management Fee** - Annual fee for fund operations (typically 2%)
- **Hurdle Rate** - Minimum return before GP earns carry
- **Clawback** - Provision to return excess carry if fund underperforms
- **TVPI** - Total Value to Paid-In (total fund value / capital called)
- **DPI** - Distributions to Paid-In (cash returned / capital called)
- **RVPI** - Residual Value to Paid-In (remaining portfolio value / capital called)
- **Waterfall** - Distribution priority structure
- **Key Person Clause** - Provision triggered if key GP leaves
