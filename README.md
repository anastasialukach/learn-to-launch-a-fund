[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](CONTRIBUTING.md)

In October 2024, after 10 years of working with tech founders and investors, I started researching how to launch a venture fund. What followed was 2,000+ hours of studying fund mechanics, GP/LP dynamics, and capital markets — supplemented by direct conversations with LPs and GPs across Asia, MENA, Europe, and the US, with each one pointing me in a new direction.

This is the compilation. Every link, podcast, data source, and framework I've gathered since.

I'm making it public because the data is clear: mega funds show declining performance — and LPs aren't sleeping on it. We're about to see even more capital flowing toward emerging managers. So now we need more of them. And better ones.

Study, create, take risks. There's nothing more attractive than competent people striving to move the world forward.

> Built by [Anastasia Lukach](https://github.com/anastasialukach) · [Venture Protocol](https://anastasialukach.github.io/venture-protocol/)

## Glossary

| Term | Definition |
|------|-----------|
| **LP (Limited Partner)** | Passive investor in the fund (provides capital) |
| **GP (General Partner)** | Active fund manager (makes investment decisions) |
| **Carried Interest (Carry)** | GP's share of fund profits, typically 20% |
| **Management Fee** | Annual fee for fund operations, typically 2% of committed capital |
| **Hurdle Rate** | Minimum return LPs must receive before GP earns carry (typically 8%) |
| **Clawback** | Provision requiring GP to return excess carry if fund underperforms |
| **TVPI** | Total Value to Paid-In — total fund value / capital called |
| **DPI** | Distributions to Paid-In — cash returned to LPs / capital called |
| **RVPI** | Residual Value to Paid-In — remaining portfolio value / capital called |
| **IRR** | Internal Rate of Return — annualized return accounting for timing of cash flows |
| **Waterfall** | Distribution priority structure (who gets paid first and how much) |
| **Key Person Clause** | Provision triggered if a named GP leaves, can pause or terminate fund |
| **Capital Call** | GP drawing down LP commitments to make investments |
| **Vintage Year** | Year a fund makes its first investment (used for benchmarking) |
| **GP Commit** | Amount GP personally invests in the fund (typically 1-3% of fund size) |
| **Side Letter** | Separate agreement giving specific LPs different terms |
| **First Close** | Initial closing where fund begins investing (before full target is raised) |
| **Final Close** | Last date for LPs to commit capital to the fund |
| **LPA** | Limited Partnership Agreement — the governing document of a fund |
| **SPV** | Special Purpose Vehicle — single-deal investment entity |
| **Rolling Fund** | Quarterly subscription fund structure (pioneered by AngelList) |
| **Fund of Funds (FoF)** | Fund that invests in other funds rather than directly in companies |
| **Pro Rata** | Right to invest proportionally in a portfolio company's future rounds |
| **NAV** | Net Asset Value — current value of fund's portfolio |
| **J-Curve** | Pattern where fund returns are negative early (fees + slow markups) before turning positive |
| **Continuation Vehicle** | New fund structure to hold assets from an older fund beyond its term |
| **OCIO** | Outsourced Chief Investment Officer — delegated investment management |
